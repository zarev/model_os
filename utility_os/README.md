# utility_os

## Setup

```bash
cd utility_os/
source osenv/bin/activate
pip install -r requirements.txt
```

## Run

### Activate env

```bash
cd utility_os/
source osenv/bin/activate
```

### Dev mode

Using textual:

```bash
textual run --dev src/utility_os/_app.py
```

### Prod mode

Directly via python:

```bash
python src/utility_os/_app.py
```
